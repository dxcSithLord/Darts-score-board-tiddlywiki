Darts Scoreboard

TiddlyWiki-inspired darts competition tracker

Welcome

A lightweight, browser-based darts scoreboard application for tracking competition scores. Built with HTML, CSS, and JavaScript - no server required.

  • Launch Scoreboard Start tracking your darts competition scores
  • Documentation View project README and setup instructions

Features

  • Multiple game types: 501, 301, 180, and Cricket
  • Track multiple players simultaneously
  • Automatic score calculation and bust detection
  • Game history tracking
  • Responsive design for mobile and desktop
  • No installation required - runs entirely in browser
